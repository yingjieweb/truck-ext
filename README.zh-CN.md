<p align="center">一款增强 Thresh 开发者工具功能的 Chrome 扩展</p>

## Features

- 💄 自定义 Thresh 开发者工具面板的样式
- 🔄 启用 Thresh 路由参数的自动编码

## Usage

- 下载存储库
- 在 Chrome 中打开[扩展程序](chrome://extensions/)页面，激活开发者模式，加载未压缩的扩展程序
- 打开 [Thresh 调试面板](http://localhost:9003/devtool/) 并通过 Chrome 浏览器访问 Thresh 开发者工具面板
